## Development Setup

在完成 `git clone` 後執行:

```bash
$ ./bin/setup
```

然後編輯專案目錄下的 `.env` 來完成組態設置。


## Testing

Run the following command to execute all test suites:

```bash
$ bundle exec rake

```
